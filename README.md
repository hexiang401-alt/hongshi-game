# 红十多人房间版 v3.1

## 运行方式

1. 解压文件夹。
2. 在文件夹内打开终端。
3. 运行：

```bash
python server.py
```

4. 房主打开：

```text
http://127.0.0.1:8000
```

5. 同一局域网的其他 5 名玩家打开：

```text
http://房主电脑IP:8000
```

例如房主电脑 IP 是 `192.168.1.23`，其他玩家打开：

```text
http://192.168.1.23:8000
```

## 已实现

- 6 人加入同一房间，真实对战。
- 房主创建房间，其他玩家输入房间码加入。
- 每个玩家只看到自己的手牌。
- 说话、出牌、不要、结算均由服务器同步。
- 出红十的人会自动公开标记为红十，但不会因此触发翻倍。
- 只有说话环节中身份被公开/揪出，底分才会翻倍。
- 玩家可以对其他未公开身份的玩家设置私人判断：未知 / 怀疑红十 / 认为平民。
- 红桃3先出。
- 下一局由上一局第1名先发牌、先说话。
- 保留出牌音效。

## 注意

这个版本使用 HTTP 轮询，不依赖第三方库，适合本地或同一 Wi-Fi 下测试。
如果要做成公网正式游戏，需要部署到云服务器，并改成 WebSocket/账号系统/断线重连/防作弊逻辑。
